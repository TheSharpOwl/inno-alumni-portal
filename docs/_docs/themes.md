#### Accessing the Portal <a name="access"></a>

- To access the Alumni Portal, open your preferred web browser and navigate to the provided URL: [www.alumniportal.com](https://github.com/Amirka-Kh/inno-alumni-portal).

    <img src="../../assets/img/1.png" alt="" width="600" height="400" style="border: 1px solid black;">

- If you are a registered user, enter your login credentials (username/email and password) in the designated fields on the login page.
- If you are accessing the portal for the first time, proceed to register for an account.

  ![Login Credentials](login_credentials.png)

**Registration**:
- If you are a new user, click on the "Register" link on the login page to create a new account.
- Provide the required information such as name, email, password, and any additional details requested during registration.

  ![Registration](registration.png)

**Single Sign-On (SSO):**
- For added convenience, users can also log in via Single Sign-On (SSO) if supported by their institution or organization.
- Simply click on the "SSO Login" option and follow the prompts provided by your institution's authentication system.

  ![SSO Login](sso_login.png)

#### User Interface Overview <a name="overview"></a>

Alumni Portal greats user with friendly interface, which consist of main menu and sidebar menu:

<img src="../../assets/img/4.png" alt="" width="600" height="400" style="border: 1px solid black;">

##### Left Sidebar Menu <a name="sidebar-menu"></a>
- Overview: Provides a summary of the user's activity, including notifications, recent events, and pending requests.
- Request Pass: Allows users to request access passes to the university for specific purposes.
- Request Elective: Enables users to request participation in elective courses offered by the university.
- Make Donations: Provides options for users to contribute financially to the university's development and initiatives.
- Account: Offers options to manage account settings, privacy preferences, and security options.

##### Main Menu <a name="main-menu"></a>
- Network: Facilitates networking opportunities by connecting users with alumni, faculty, and professionals.
- Courses: Offers access to additional studies and professional development courses tailored for graduates.
- Pass: Allows users to request passes for accessing university facilities and premises.
- Donations: Provides options for users to support the university through financial contributions.

With intuitive navigation and clearly defined sections, users can easily explore and utilize the various features and functionalities offered by the Alumni Portal. Whether connecting with fellow alumni, exploring educational opportunities, or contributing to the university's growth, the user interface provides a seamless experience for alumni engagement and interaction.

#### Admin Panel <a name="admin"></a>

To access the admin panel, select the "Login as Admin" section on the login page.

<img src="../../assets/img/17.png" alt="" width="600" height="400" style="border: 1px solid black;">

Admins are responsible for managing requests and appointments. Requests are categorized by type (elective requests, appointment requests, etc.). In each requests panel, admins can view a list of requests.

<img src="../../assets/img/18.png" alt="" width="600" height="400" style="border: 1px solid black;">

Admins have the authority to approve or reject requests as they see fit.

<img src="../../assets/img/19.png" alt="" width="600" height="400" style="border: 1px solid black;">

Additionally, admins manage courses and events, where they can modify descriptions, formats, course names, instructor names, and even delete event fields.

<img src="../../assets/img/20.png" alt="" width="600" height="400" style="border: 1px solid black;">

Admins can also add other admins through a separate field.

<img src="../../assets/img/21.png" alt="" width="600" height="400" style="border: 1px solid black;">
